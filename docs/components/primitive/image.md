# UImage

## Properties

### src
* Type:String

### alt
* Type:String

### rounded
* type: boolean
* default: false

### caption
* type: String
* default: clean

### CaptionAlign
* type: String
* default: clean
* available values: top, left, right,bottom,center

### TextColor
* type: String
* default: clean
* available values: #CD5C5C,#E9967A,...
